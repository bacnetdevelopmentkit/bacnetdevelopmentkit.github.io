BACnet MS/TP Development Kit - Contents of the kit include the following:

  • BACnet MS/TP Development Board (loaded with the BACnet Stack firmware)
  • 5.0Vdc Power Supply
  • USB cable
  • BACnet Stack and Application Software
  • Manual on USB drive
  • Examples and Sample Code on USB drive
  • Data Sheets on USB drive
  • Schematic, PCB, and BOM on USB drive
  • Printed Overview Page
